# Turn-Based Combat Game (C++)

**Описание:**  
Пошаговая консольная RPG на C++ с картой 15x15, где игрок сражается с гоблинами.  
Игра демонстрирует паттерны проектирования:  

- **Стратегия (Strategy):** действия персонажей (атака, огненный шар, блок) реализованы как отдельные стратегии.  
- **Фабрика (Factory):** класс `ActionFactory` создает действия по имени, упрощая расширение игры.  

**Функциональность:**  
- Игрок и гоблин имеют здоровье, ману и броню.  
- Игрок выбирает действия:  
  1. Атака — наносит физический урон  
  2. Огненный шар — наносит магический урон, тратит ману  
  3. Блок — уменьшает входящий урон вдвое  
- Гоблин может атаковать или блокировать.  
- Карта 15x15 с объектами:  
  - 1 — игрок  
  - 2 — враг  
  - 3 — здоровье  
  - 4 — мана  
  - 5 — золото  
  - 9 — стены  
- Управление: `W` (вверх), `A` (влево), `S` (вниз), `D` (вправо)

**Запуск:**  
1. Скомпилируйте файл:
bash
`g++ -o game GameTurn-Based-combat.cpp`

2. Запустите
`./game`
